### passive-interface

라우팅 업데이트 발신(송신)을 차단함<br/>
주로 LAN이나 WAN상에서 원하지않는 네트워크 정보 전송 차단<br/>

#### R4에서 보안 목정상 f1/0 인터페이스 정보 송출을 차단하기

#### R4

conf t<br/>
router rip<br/>
passive-interface f0/0<br/>

#### R4

conf t<br/>
int f0/0<br/>
ip summary-address rip 10.0.0.0 255.255.252.0<br/>
ip summary-address rip 192.168.0.0 255.255.254.0<br/>
ip summary-address rip 172.0.0.0 255.254.0.0<br/>

<br/>
