#### Server 

기본 모드이며 vlan 생성, 삭제 vlan 이름 병경 및 vtp 정보를 다른 스위치에 전송한다<br/>
conf t<br/>
vtp mode server <br/>
vpt domain cisco<br/>
<br/>
#### Client

vlan을 생성하거나 삭제할 수 없으며, 자신의 vtp 정보를 다른 스위치에 전송한다<br/>
conf t<br/>
vtp mode client<br/>

#### Transparent

자신의 vtp 정보를 다른 스위치에 전송하지 않으며, 다른 스위치에서 받은 정보와 일치시키지 않는다<br/>
conf t<br/>
vtp mode transparent<br/>
